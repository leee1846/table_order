┌─────────────────────────────────────────────────────────────────────────────┐
│                           전체 흐름 요약                                      │
└─────────────────────────────────────────────────────────────────────────────┘

1. 다이얼로그 열기
   └─► API 호출 (카테고리 + 메뉴 데이터)
       └─► MenuSelectionView 표시

2. 메뉴 선택
   ├─► 옵션 없는 메뉴 클릭
   │   └─► 바로 selectedMenus에 추가
   │
   └─► 옵션 있는 메뉴 클릭
       └─► OptionSelectionView로 전환
           ├─► 옵션 선택
           ├─► 메뉴 수량 설정
           └─► 추가하기 클릭
               └─► selectedMenus에 추가
                   └─► MenuSelectionView로 돌아감

3. 선택된 메뉴 관리 (MenuSelectionView)
   ├─► 수량 변경
   ├─► 삭제
   └─► 추가하기 클릭
       └─► onAdd 콜백 호출
           └─► 다이얼로그 닫기


┌─────────────────────────────────────────────────────────────────────────────┐
│                           상태 관리 흐름                                      │
└─────────────────────────────────────────────────────────────────────────────┘

State Variables:
├── viewMode: 'menu' | 'option'
│   └─► 현재 표시할 화면 결정
│
├── selectedCategory: number | null
│   └─► 선택된 카테고리 (MenuSelectionView에서 사용)
│
├── selectedMenu: MenuboardMenu | null
│   └─► 옵션 선택 중인 메뉴 (OptionSelectionView에서 사용)
│
├── selectedMenus: SelectedMenuWithOptions[]
│   └─► 최종적으로 추가할 메뉴 목록
│
├── selectedOptions: Map<number, number>
│   └─► 옵션별 수량 (OptionSelectionView에서 사용)
│       Key: optionSeq, Value: quantity
│
└── menuQuantity: number
    └─► 메뉴 수량 (OptionSelectionView에서 사용)


┌─────────────────────────────────────────────────────────────────────────────┐
│                           데이터 타입 구조                                    │
└─────────────────────────────────────────────────────────────────────────────┘

MenuboardCategory
├── categorySeq: number
├── categoryName: string
└── menuInfoList: MenuboardMenu[]

MenuboardMenu
├── menuSeq: number
├── menuName: string
├── menuPrice: number
└── optionGroupList: MenuboardOptionGroup[]

MenuboardOptionGroup
├── optionGroupSeq: number
├── optionGroupName: string
├── minQuantity: number
├── maxQuantity: number
├── isMultipleSelectable: boolean
├── isOptionQuantitySelectable: boolean
└── optionList: MenuboardOption[]

MenuboardOption
├── optionSeq: number
├── optionName: string
├── optionPrice: number
├── isOutOfStock: boolean
└── isDeleted: boolean

SelectedOption extends MenuboardOption
└── selectedQuantity: number

SelectedMenuWithOptions
├── menu: MenuboardMenu
├── selectedOptions: SelectedOption[]
└── quantity: number
